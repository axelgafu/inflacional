# Project Glossary

Standard terminology used across technical and user documentation for the **inflacional** project.

## Terms

### Banxico SIE API
The official Bank of Mexico Service of Information of Economic (SIE) REST API.

### INPC (√çndice Nacional de Precios al Consumidor)
The National Consumer Price Index, used to measure inflation in Mexico.

### ADR (Architecture Decision Record)
A document that captures an important architectural decision made along with its context and consequences.

### Provider Factory
The design pattern used to instantiate either a real API client or a mock client based on the environment.

### BDD (Behavior-Driven Development)
The development methodology used to ensure project quality through acceptance tests written in plain language.
